Shifu Website
=============



Local Development
-----------------

Install ``guard`` and ``nanoc``:

    $ gem install guard
    $ gem isntall nanoc

To monitor the project and re-compile upon changes, use ``guard``:

    [/path/to/shifu-website] $ guard

To serve the website locally, use ``nanoc view``:

    [/path/to/shifu-website] $ nanoc view
